# 1.2 UNIX 体系
kernel ：内核
systen call : 系统调用，用作内核的接口
而公用函数库，shell，应用程序都是建立在系统调用上的
shell： 特殊的应用程序。程序语言，程序解释语言和xxx的统称
system utility : 系统实用程序

* Linux是gnu操作系统使用的内核，GNU/Linux操作系统，一般简称为linux，但是严格意义上是不对的，但是介于“操作系统”的两重含义，可以认为其没有错误。*
	- * 对硬件：抽象硬件，管理虚拟接口*
	- * 对应用软件：使用标准接口运行应用或完成任务，管理应用（包括加载，启动，切换，调度以及销毁等）*
* Linux内核：提供了系统的底层实现机制*
* GNU系统：提供了封装库，标准接口，以及调用这些接口的上层工具*
