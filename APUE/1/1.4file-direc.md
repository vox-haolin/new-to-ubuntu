# 1.4 file and directory

## 1. file system
UNIX文件系统是文件和目录的一种层次结构
所有的起点root（根），字符 “/”
目录就是一个文件夹，任何目录都包含文件名和文件属性
* 文件属性：文件类型，文件大小，文件所有者/组，文件权限，修改时间等*
可用指令 : `stat` `fstat`

## 2. filename
* 只有`/` 或者` `空字符，不能用来作文件名*
`/`用来进行分隔构成路径名的各个文件名，空字符` `用来终止一个路径名
每创建一个新目录就会自动建立两个新文件夹.和.. ， .指向当前文件夹,..指向父目录。但是root的.和..都指向本身

## 3. pathname
（/xxx/）xx/xx，表示一个具体的文件或目录
absolute pathname：以/（即根节点）开头的，是绝对路径
relative pathname：相对路径
* 文件系统的根的名字（/）是一个比较特殊的绝对路径，他不包含文件名，但是确实一个目录*
